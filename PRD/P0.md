# ABA Edu Portal

## One-liner
발달지연 아동 부모를 위한 ABA 교육 콘텐츠 포털 — 매거진, 가이드, 교육 앱을 한곳에서.

## Audience
발달장애/발달지연 자녀(2-10세)를 둔 한국 부모.
ABA 초보, 스마트폰 중심, 짧고 핵심적인 정보 선호.

## Core Features
- 콘텐츠 피드 (카드뉴스/만화/아티클/인포그래픽/영상)
- 카테고리별 필터링 (언어, 사회성, 일상생활, 행동 관리, ABA 기초)
- 콘텐츠 상세 (타입별 렌더링 — 이미지 시퀀스, 리치텍스트, 영상 임베드)
- 교육 앱 섹션 (실사용 2개 + Coming Soon 1-3개, 태블릿 전체화면 실행)
- Notion API 기반 콘텐츠 관리
- SEO (메타태그, OG, sitemap)
