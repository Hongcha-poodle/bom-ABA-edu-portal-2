## Style
- Tone: Playful, warm, approachable — 기존 구현 유지
- Display Font: Fredoka (제목)
- Body Font: Inter + Pretendard (본문)
- Colors: 기존 globals.css 테마 토큰 그대로 사용
  - Primary: Lavender Pink (#E879F9 계열)
  - Secondary: Mint Green (#5EEAD4 계열)
  - Accent: Soft Yellow (#FEF08A 계열)
  - Tertiary: Sky Blue (#BAE6FD 계열)
  - 각 섹션 배경: 기존 패턴 유지 (Hero=#CCFBEA, 매거진=#FEF9C3, 가이드=#E0F2FE, 도구=#99F6D4)
- Card Style: rounded-3xl, white bg, shadow-soft, hover lift
- 신규 페이지 배경색:
  - Category: 해당 카테고리 컬러 연하게 (예: 언어=#FAE8FF, 사회성=#CCFBEA)
  - Content Detail: white
  - Apps List: #F0ABFC (primary-300)
  - About: 홈 Hero와 동일 그라디언트

## Responsive
- Mobile-first (기존과 동일)
- Breakpoints: sm(640), md(768), lg(1024), xl(1280)
- 카드 그리드: 1열(mobile) → 2열(sm) → 3열(lg) → 4열(xl, 홈 매거진만)

## Done When
- [ ] Category 페이지: 카테고리 헤더 + 필터 칩 + 카드 그리드가 브라우저에서 확인됨
- [ ] Content Detail 페이지: 3가지 타입(이미지 시퀀스, 리치텍스트, 영상) 각각 목업 데이터로 표시됨
- [ ] Apps List 페이지: 실사용 2개 + Coming Soon 카드가 그리드로 표시됨
- [ ] App Detail 페이지: 설명 영역 + 전체화면 실행 버튼 레이아웃 확인됨
- [ ] About 페이지: 정적 콘텐츠 레이아웃 확인됨
- [ ] 404 페이지: 중앙 정렬 에러 메시지 + 홈 링크 확인됨
- [ ] 모든 신규 페이지가 320px~1920px에서 깨짐 없이 표시됨
- [ ] 기존 홈 페이지 스타일과 톤 일관성 유지됨
